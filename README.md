# Sri Senthil Silks Website

A modern business website developed for **Sri Senthil Silks**, a traditional silk saree store in Kanchipuram.

## Live Website

https://silk-weaver-stories.lovable.app

## GitHub Repository

https://github.com/atchaya24112007-a11y/silk-weaver-stories

## Features

* Responsive and mobile-friendly design
* Premium silk saree showcase
* Direct WhatsApp contact integration
* Google Maps location integration
* Business information and contact details
* Clean and user-friendly interface

## Technologies Used

* React
* TypeScript
* TanStack Start
* Tailwind CSS
* Lovable AI
* GitHub

## Project Overview

This website was developed to help Sri Senthil Silks establish an online presence and make it easier for customers to explore products, contact the business, and locate the store.

## Key Integrations

* WhatsApp Business Contact
* Google Maps Navigation
* Mobile Responsive Layout

## Deployment

The project is currently deployed and accessible online through Lovable hosting.
